<objective>
Generate comprehensive roadmap showing status across all focus groups and active implementations.
Updates MASTER-ROADMAP.md and syncs with main development channel canvas.
</objective>

<process>

## Step 1: Gather Focus Group Data

```bash
REPO_NAME=$(basename $(pwd))
TIMESTAMP=$(date -u +"%Y-%m-%d %H:%M:%S UTC")

echo "📊 Generating WGSD roadmap for ${REPO_NAME}"

# Count focus groups and concepts
FOCUS_GROUPS_COUNT=0
TOTAL_CONCEPTS=0
READY_CONCEPTS=0

if [ -d .planning/focus-groups ]; then
    FOCUS_GROUPS_COUNT=$(find .planning/focus-groups -maxdepth 1 -type d | grep -v "^\\.planning/focus-groups$" | wc -l)
fi
```

## Step 2: Scan Focus Groups

For each focus group directory:

```bash
for focus_group in .planning/focus-groups/*/; do
    if [ -d "$focus_group" ]; then
        FG_NAME=$(basename "$focus_group")
        
        # Count concepts in this focus group
        if [ -d "$focus_group/concepts" ]; then
            FG_CONCEPTS=$(find "$focus_group/concepts" -name "*.md" | wc -l)
            TOTAL_CONCEPTS=$((TOTAL_CONCEPTS + FG_CONCEPTS))
        else
            FG_CONCEPTS=0
        fi
        
        # Parse ROADMAP.md for ready concepts (if exists)
        if [ -f "$focus_group/ROADMAP.md" ]; then
            FG_READY=$(grep -c "🚀 Ready for Implementation" "$focus_group/ROADMAP.md" 2>/dev/null || echo 0)
            READY_CONCEPTS=$((READY_CONCEPTS + FG_READY))
        else
            FG_READY=0
        fi
        
        echo "📁 ${FG_NAME}: ${FG_CONCEPTS} concepts (${FG_READY} ready)"
    fi
done
```

## Step 3: Scan Active Implementations

```bash
IMPLEMENTATIONS_COUNT=0
if [ -d .planning/active-implementations ]; then
    IMPLEMENTATIONS_COUNT=$(find .planning/active-implementations -maxdepth 1 -type d | grep -v "^\\.planning/active-implementations$" | wc -l)
fi

echo "🚀 Active implementations: ${IMPLEMENTATIONS_COUNT}/4"
```

## Step 5: Detect Cross-References

```bash
# Search for cross-references between focus groups
echo "🔍 Scanning for cross-focus-group references..."

if [ -d .planning/focus-groups ]; then
    # Look for references to other focus groups in concept files
    find .planning/focus-groups -name "*.md" -exec grep -l "focus-groups/" {} \; 2>/dev/null | head -5
fi
```
